# 사용자 여정

## 핵심 사용자

- 서울 소형 상가를 검토하는 개인 투자자
- 중개사 설명은 들었지만 숫자로 다시 검증하고 싶은 사용자
- 현장 답사 전에 빠른 1차 필터가 필요한 사용자

## 사용자 흐름

### 1. 유입

사용자는 `좋아 보이는 구` 또는 `검토 중인 매물`을 이미 하나 들고 들어옵니다.  
문제는 정보 부족이 아니라, 좋은 이야기만 먼저 들었다는 점입니다.

### 2. 1차 필터

사용자는 3분 진단이나 내 매물 검토에서 다음을 입력합니다.

- 구
- 희망 가격 또는 평당가
- 보유 기간

### 3. 보류 사유 확인

시스템은 총점, 리스크 유형, 보류 사유를 먼저 보여줍니다.

### 4. 대체 후보 비교

사용자는 지금 보던 후보를 바로 사는 대신, 더 낮은 위험 후보를 비교합니다.

### 5. 현장 확인

사용자는 현장 체크리스트를 들고 실제로 확인할 질문을 가져갑니다.
